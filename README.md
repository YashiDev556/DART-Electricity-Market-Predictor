Utilized a two-branch neural network for predicting electricity prices using ECMWF ERA5 reanalysis data, federal funds rate data, and real-time electricity prices. Focused on predicting prices in the LZ_Houston Area (Houston Loading Zone) of the Electric Reliability Council of Texas (ERCOT). 


Project for ORF455 by John Geiss '27, Henry Laufenberg '26, and Jan Iyer '27
